# NTP DIG PING MORE

Hi folks,

This project started because I couldn't find a reliable NTP client.
This app is intended for ALL public.
My intent is to provide an easy to use, yet powerful tool to make simple network tests.


The aim is to have a versatile network toolbox that respects Android standards.
As far as I see, it is quite easy to get a rooted device and move on with it.
Nevertheless, a tool respecting Android standards, working on non-rooted devices, that can be run simply felt more perennial.
Here I went for "let's do it by the book"


To be frank, I was really fed with all the apps out there containing ads.
When you are doing network checks, those ads add
noise and are ideal to bias tests we intend to do.

The app is intended to work on closed networks so it doesn't need internet access.
The only feature that does is GoogleTimeSync, I'll come back to that later.

## Main takeaway

- [x] Free app (no ads)
- [x] Works in closed network
- [x] Bulk actions available

## Tools

- NTP
- DIG
- PING
- PORT SCANNER
- TRACEROUTE
- GOOGLE TIMESYNC (Supports ProxyPAC)
- HTTPS CERTIFICATE INSPECTOR (Supports ProxyPAC)
