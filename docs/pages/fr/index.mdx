# NTP DIG PING MORE

Bonjour à tous,

Ce projet a commencé parce que je ne trouvais pas de client NTP fiable.

L'objectif est d'avoir une boîte à outils réseau polyvalente qui respecte les normes Android.
Dans la mesure où je vois les choses, il est assez aisé de rooter un appareil et de procéder ainsi.

Néanmoins, un outil respectant les normes Android, fonctionnant sur les appareils non rootés, qui peut être exécuté simplement m'a semblé plus pérenne.
Je suis parti sur une approche "respectons le cahier des charges".

Ma conclusion est que cette application s'adresse à TOUS le public.
Mon intention était de fournir un outil facile à utiliser, mais puissant, pour effectuer des tests réseau simples.

Pour être franc, j'étais vraiment las de toutes les applications contenant des publicités. 
Sans compter que cela rajoute du bruit quand on fait des tests réseau,
en réseau fermé de surcroît.

L'application est conçue pour fonctionner sur des réseaux fermés, elle n'a donc pas besoin d'accès Internet.
La seule fonctionnalité qui en a besoin est GoogleTimeSync, j'y reviendrai plus loin.

## Points principaux

- [x] Application gratuite (pas de publicités)
- [x] Fonctionne en réseau fermé
- [x] Actions en lots disponibles
- [ ] Pas besoin d'accès interne
