# NTP DIG PING MORE

Hi folks,

This project started because I coundn't find a reliable NTP client.

The aim is to have a versatile network toolbox that respect android standard.
as far as I see it is quite easy to get a rooted device and move on with it.

Nevertheless a tool respecting Android standard, working on non-rooted devices, that can be ran simply felt more perennial
Here I went for "let's do it by the book"

Main takeaway is this app is intended to ALL public,
my intent was to provide an easy to use, yet powerfull tool to make network tests.

To be frank I was really fed with all the app out there containing ads,
among the fact it's ugly, when you are doing network check those ads adds
noise and are ideal to bias tests we intend to do.

The app is intended to work on closed networks so it doesn't need internet access,
the only feature that does is GoogleTimeSync, i'll come back to that later.

## Main takeaway

[x] Fee app (no ads)
[x] Works in closed network
[x] Bulk actions available

## Tools

- NTP
- DIG
- PING 
- PORT SCANNER
- TRACEROUTE
- GOOGLE TIMESYNC (Supports ProxyPAC)
- HTTPS CERTIFICATE INSPECTOR (Supports ProxyPAC)
- DEVICE INFOS
